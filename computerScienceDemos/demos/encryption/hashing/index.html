Hashing

Overview

Hashing is a one-way function that given an input will create a unique output. This output is consistent, given the same input, you get the same output. A Hashing function has a collision rate, The collision rate of a given hashing function is the chance that two different inputs give the same output.

Keywords

  • One-way
  • Collision Rate
  • Unique Output
